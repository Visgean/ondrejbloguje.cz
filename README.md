## Blogs based on uBlog

* <http://djangoblog.cz>
* <http://pythonblog.cz>


## Usage

* __./serve.sh__ - run jekyll development server on __0.0.0.0:4000__
* __./build.sh__ - build site to **_site** directory
* __./deploy.sh__ - git push to all remotes
